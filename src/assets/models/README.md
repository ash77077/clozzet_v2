# 3D Clothing Models for Custom Apparel Feature

This directory should contain 3D models in GLB/GLTF format for realistic clothing visualization.

## Required Files

Place the following files in this directory:
- `tshirt.glb` - T-shirt 3D model
- `polo.glb` - Polo shirt 3D model
- `hoodie.glb` - Hoodie 3D model

## Model Requirements

For best results, your 3D models should:
- Be in GLB or GLTF format
- Have proper UV mapping for textures
- Be centered at origin (0, 0, 0)
- Be reasonable size (will be auto-scaled)
- Have clean topology
- Include basic materials

## Fallback Behavior

If models are not found, the system will automatically use procedurally generated clothing shapes as a fallback. This ensures the application works even without the GLB files.
